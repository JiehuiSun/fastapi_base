# FastApi Base

*fastapi + mongodb*

## Clone
```shell
$ git clone https://github.com/JiehuiSun/fastapi_base.git
```

## Installation
### PIP
```shell
$ python -m venv .venv &&source .venv/bin/activate &&pip install -r ./requirements.txt
```

### Poetry
```shell
$ poetry new
$ poetry install
```

### Start
```shell
$ uvicorn app.main:app --reload
```
